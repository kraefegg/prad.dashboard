# Dashboard PRAD — UFV Rio do Peixe I e II

Dashboard interativo (HTML/Chart.js) do monitoramento PRAD — Caraúbas-PB, Junho/2026.
Bragagnoli Ambiental / EKORA Consultoria Ambiental.

## Deploy (GitHub Pages)
1. Criar repo → subir `index.html`.
2. Settings → Pages → Branch: `main` /root → Save.
3. URL: `https://<usuario>.github.io/<repo>/`

## Conteúdo
- Visão Geral (KPIs, sobrevivência, indicadores)
- Espécies & Sobrevivência
- Fatores Supervenientes + Matriz de Risco
- Manejo & Ações
- Áreas & Cronograma

Arquivo único, sem dependências externas de build (CDN Chart.js).
